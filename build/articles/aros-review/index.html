Operating System Review - AROS

AROS (AROS Research Operating System, yet another recursive acronym) is an Operating System aiming to be API compatible with Amiga 3.x systems. It is free software, however it does have licensing issues.. For the review, I have chosen the Icaros Desktop (version v2.1.2) distribution as it’s pre-configured for the PC platform.

I personally have a minor interest for the Amiga systems due to the Demoscene and mod tracking but I’ve never owned one or have any interest in the games available.
